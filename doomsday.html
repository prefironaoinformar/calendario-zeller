Doomsday Rule

Explicação detalhada da Doomsday Rule para o calendário Gregoriano

Conceito Básico

A Doomsday Rule, desenvolvida por John Conway, é um método mental para determinar o dia da semana de qualquer data no calendário gregoriano. A ideia é que certos dias do mês (os "doomsdays") caem no mesmo dia da semana em um dado ano.

Passo 1: Obtendo a Âncora do Século

Para o calendário gregoriano, a âncora do século é calculada por:

S = (5 × (C mod 4) + 2) mod 7

Onde C é o número do século (por exemplo, para 2025, C = 20).
Exemplo: Para o século 21 (C = 20):

S = (5 × (20 mod 4) + 2) mod 7
  = (5 × 0 + 2) mod 7
  = 2 mod 7
  = 2   (2 corresponde a Terça-feira, considerando 0 = Sábado, 1 = Domingo, ...)

Passo 2: Obtendo a Âncora do Ano

Utilize os dois últimos dígitos do ano (YY) e calcule:

a = ⌊YY/12⌋
b = YY mod 12
c = ⌊b/4⌋
Doomsday do ano = (S + a + b + c) mod 7

Exemplo: Para 2025 (use YY = 25):

a = ⌊25/12⌋ = 2
b = 25 mod 12 = 1
c = ⌊1/4⌋ = 0
Doomsday do ano = (2 + 2 + 1 + 0) mod 7 = 5 mod 7 = 5

Interpretação: 5 corresponde a Quinta-feira, segundo a convenção (0 = Sábado, 1 = Domingo, 2 = Segunda, 3 = Terça, 4 = Quarta, 5 = Quinta, 6 = Sexta).

Passo 3: Âncora do Mês

Os doomsdays mensais no calendário gregoriano são fixos:

  • Janeiro: 3 (se não bissexto) ou 4 (se bissexto)
  • Fevereiro: 28 (se não bissexto) ou 29 (se bissexto)
  • Março: 14
  • Abril: 4
  • Maio: 9
  • Junho: 6
  • Julho: 11
  • Agosto: 8
  • Setembro: 5
  • Outubro: 10
  • Novembro: 7
  • Dezembro: 12

Exemplo Detalhado – Data: 18/02/2025

  1. Ajuste para Fevereiro: Como fevereiro é especial, usamos 28 em anos não bissextos (2025 não é bissexto). Portanto, o doomsday de fevereiro é 28.
  2. Âncora do Século: Para 2025, C = 20: 
    S = (5 × (20 mod 4) + 2) mod 7
  = (5 × 0 + 2) mod 7
  = 2
    (2 corresponde a Terça-feira.)
  3. Âncora do Ano: Use YY = 25: 
    a = ⌊25/12⌋ = 2
b = 25 mod 12 = 1
c = ⌊1/4⌋ = 0
Doomsday do ano = (S + a + b + c) mod 7
                = (2 + 2 + 1 + 0) mod 7
                = 5 mod 7 = 5
    (5 corresponde a Quinta-feira.)
  4. Ajuste para o Mês: Para fevereiro, o doomsday é 28. A data em questão é 18, que está 10 dias antes de 28. 
    Diferença = 18 - 28 = -10
Ajuste: (-10) mod 7 = -3 mod 7 = 4 (ou seja, retroceda 3 dias a partir do doomsday do ano)
  5. Cálculo Final: Partindo do doomsday do ano (quinta-feira, 5), retrocedendo 3 dias: 
    Quinta (5) → Quarta (4) → Terça (3) → Segunda (2)  [ou, usando a convenção: 5 - 3 = 2]
    Dependendo da convenção, ajuste para obter o dia correto. Neste exemplo, o cálculo indica que a data 18/02/2025 cai em Terça-feira (se a retrocessão de 10 dias for interpretada corretamente).
  6. Interpretação: Após ajustar a diferença, o dia da semana obtido é Terça-feira.
